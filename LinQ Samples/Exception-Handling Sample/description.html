Indexeurs, exemple

Cet exemple montre comment les classes C# peuvent déclarer des indexeurs pour fournir un accès de type tableau aux classes.

Note de sécurité

Cet exemple de code est fourni pour illustrer un concept et ne doit pas être utilisé dans des applications ou des sites Web, car il peut ne pas illustrer les pratiques de programmation les plus sûres. Microsoft n'assume aucune responsabilité pour tout dommage indirect ou consécutif en cas d'utilisation de l'exemple de code à des fins autres que celles prévues.

Pour générer et exécuter l'exemple Indexeurs dans Visual Studio

  1. Dans l'Explorateur de solutions, cliquez avec le bouton droit sur le projet Indexers, puis cliquez sur Propriétés.

  2. Ouvrez le dossier Propriétés de configuration, puis cliquez sur Déboguer.

  3. Dans la propriété Arguments de la ligne de commande, entrez ..\..\Test.txt.

  4. Cliquez sur OK.

  5. Dans le menu Déboguer, cliquez sur Exécuter sans débogage.

Pour générer et exécuter l'exemple Indexeurs à partir de la ligne de commande

  1. Pour compiler l'exemple de programme, tapez le texte suivant à l'invite de commandes : 

    csc indexer.cs

  2. L'exemple de programme inverse les octets dans un fichier fourni en tant qu'argument de la ligne de commande. Par exemple, pour inverser les octets dans Test.txt et voir le résultat, exécutez les commandes suivantes : 

    indexers Test.txt
type Test.txt

  3. Pour restaurer l'état initial du fichier après l'inversion, réexécutez le programme sur le même fichier.
